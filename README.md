# BrainSys

## Hardware

* Arduino Nano (ATMEGA328P)
* OLED 128x32  Display module IIC Communicate
* Piezo Beep
* Buttons
* LEDs

## Used ATMEGA Pins

* d2 - button - team 1
* d3 - button - team 2
* d4 - button - team 3
* d5 - button - team 4
* d6 - button - reset
* d7 - button - start

* d9 - beep

* a0 - led - team 1
* a1 - led - team 2
* a2 - led - team 3
* a3 - led - team 4
* a4 - OLED - SDA
* a5 - OLED - SCK
